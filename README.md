# 🤖 Telegram UserBot - Рассылка сообщений от вашего лица

Мощный юзербот для автоматической рассылки сообщений в Telegram через вашего бота-помощника. Отправляйте сообщения в любые чаты от своего имени с удобным управлением!

---

## ✨ Возможности

- 📨 **Отправка от вашего лица** - сообщения приходят именно от вас
- 🎯 **Массовая рассылка** - отправка в несколько чатов одновременно
- ⏱ **Интервалы** - настройка пауз между сообщениями
- 🔄 **Автоповтор** - циклическая отправка с заданным периодом
- 📝 **Несколько сообщений** - отправка цепочки сообщений подряд
- 🎮 **Управление через бота** - все команды через Telegram
- ☁️ **24/7 работа** - бесплатный хостинг на Render.com
